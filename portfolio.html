This project is a proof of concept of person/group matching. Given a set of criteria the user can answer questions and be matched with other users who answered similarly. This could be for friends, game groups, or any other number of targets.

To Do List with Randomizer

This project is a work in progress that is meant to help people keep track of different activities. The aspect that sets this apart from any other organizer is that there is a random button in the case of a list of shows, books, or movies to watch or read but the user is undecided in which order to do so, this will alleviate that stress.
